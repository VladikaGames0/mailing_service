# Сервис управления рассылками

## Описание проекта
Веб-приложение на Django для управления email-рассылками. Позволяет создавать рассылки, управлять получателями и сообщениями, отслеживать статистику отправок.

## Функциональность
- ✅ Управление получателями (CRUD)
- ✅ Управление сообщениями (CRUD)
- ✅ Управление рассылками (CRUD)
- ✅ Автоматическое обновление статусов рассылок
- ✅ Отслеживание попыток отправки
- ✅ Аутентификация и регистрация пользователей
- ✅ Личный кабинет с профилем
- ✅ Разграничение прав доступа (владельцы, менеджеры)
- ✅ Кеширование главной страницы
- ✅ Отправка рассылок через интерфейс и командную строку

## Технологии
- Python 3.11
- Django 5.2
- Bootstrap 5
- SQLite
